Thank you for installing GPGTools
Version: 20110628

In order to use OpenPGP on OS X you can chose to install / update to:

* MacGPG 2.0.17-9 (incl. gpg-pinentry and gpg-agent)
* GPGMail 1.3.3 (for OS X 10.6 Mail.app)
* GPGMail 1.2.0 (for OS X 10.5 Mail.app)
* GPGMail 1.1.2 (for OS X 10.4 Mail.app)
* GPG Keychain Access 0.8.10 (managing GPG keys)
* GPGServices 1.6 (OS X service menu for GPG)
* GPGPreferences 0.6 (GPGTools preferences, some autofix only)
* Enigmail 1.1.2 (for Mozilla Thunderbird 3) [if Thunderbird is detected]
* Enigmail 1.2 (for Mozilla Thunderbird 5) [not installed by default]
* MacGPG 1.4.11-6 (simple MacGPG version) [not installed by default]

Have a look at http://gpgtools.org for further information.

Note: please close Mail.app first
