Thank you for installing GPGTools 201108XX

In order to use OpenPGP on OS X you can chose to install:

(have a look at http://gpgtools.org for further information)

* GPG Keychain Access 0.8.13
  * Description: Managing OpenPGP keys
  * Requirements: OS X >= 10.5
* GPGServices 1.6
  * Description: OpenPGP Service Menu for files and text selections
  * Requirements: OS X >= 10.6
* GPGMail 2.0.a2 / 1.3.3 / 1.2.0
  * Description: OpenPGP integration for Mail.app
  * Requirements: OS X 10.7 / 10.6 / 10.5
* GPGToolsPreferences 0.6
  * Description: OpenPGP system preferences
  * Requirements: OS X >= 10.5
* Enigmail 1.2.1 / 1.1.2
  * Description: OpenPGP integration for Thunderbird
  * Requirements: Thunderbird 5.x / 3.x
* MacGPG 2.0.17-9 / 1.4.11-6
  * Description: OpenPGP core implementation
  * Requirements: OS X >= 10.5, Intel / OS X >= 10.5, Universal
